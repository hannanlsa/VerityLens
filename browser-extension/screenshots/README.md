## Tips / 提示

- Use Chrome's built-in screenshot tool: DevTools → ⋮ → Capture screenshot
- Or use the `Ctrl+Shift+S` shortcut in Firefox
- Make sure the extension UI is clearly visible
- Use Chinese search results for zh-CN market
- Consider adding English screenshots for international users
