## Detailed Description

🛡️ VerityLens — Your Truth Lens for Search Results

Automatically annotate the credibility of every search result, helping you spot ads, SEO farms, and trustworthy content at a glance. Annotations are based on big data AI analysis and are for reference only.

🔍 Supported Search Engines
Baidu · 360 Search · Sogou · Google · Bing · DuckDuckGo

📊 5-Level Credibility Rating
• 🟢 High — Official/academic/authoritative sources
• 🟡 Medium — Personal blogs/moderately credible
• 🟠 Partial — Credible but with concerns
• 🔴 Abnormal — Likely ads/SEO farms/sponsored content
• ⚪ Unverified — Cannot determine

🔒 4 Verification Modes
• Local Only — Zero network requests, zero privacy risk, works out of the box
• Smart Switch (Recommended) — Lightweight → local, complex → AI, automatic selection
• Cloud Only — All verification via AI models, highest accuracy
• Self-hosted Docker — Run your own AI model on your PC or NAS, fully private, for advanced users

🐳 Self-hosted Docker (Advanced)
If you have a GPU and Docker, run your own AI model on your PC or NAS:
• Fully private — data never leaves your network

🌐 Bilingual Translation (Emergency)
• One-click side-by-side translation for foreign-language pages
• Supports 6 UN official languages: Chinese/English/French/Spanish/Russian/Arabic
• ⚠️ If the website has an official language switch, please use that first. This translation is for emergency use only.

🧠 Smart Features
• Auto-detect ad/marketing/SEO characteristics in search results
• Image text recognition for verification
• Voice content analysis for verification
• Right-click context menu for quick verification
• Adjustable confidence threshold

⚡ Quick Start
1. Install and visit any supported search engine
2. Results are automatically annotated with color bars
3. Hover to see detailed scores and reasons
4. Want higher accuracy? Select a free AI model in Settings

🛡️ Privacy Protection
• Local mode: Zero data collection, zero network requests
• Cloud mode: Only sends search result text to your chosen AI provider
• Self-hosted Docker: Data never leaves your network
• API keys stored locally in your browser — we never collect them
