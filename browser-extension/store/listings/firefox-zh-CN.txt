## Name
VerityLens · 真实透镜

## Summary (250 chars max)
信息可信度标注 + 外文页面双语翻译，帮你识别广告、SEO农场和可信内容。支持6大搜索引擎，9家免费AI模型，4种隐私模式，可自建Docker本地AI。

## Description

🛡️ VerityLens = 真实透镜

在搜索引擎结果页自动标注每条结果的可信度，帮你一眼识别广告、SEO农场和真实内容。标注来源为大数据AI分析，仅供参考。

🔍 支持的搜索引擎
百度 · 360搜索 · 搜狗 · Google · Bing · DuckDuckGo

📊 5级可信度标注
• 🟢 高置信 — 官方/学术/权威来源
• 🟡 中等 — 个人博客/中等可信来源
• 🟠 部分验证 — 有疑点
• 🔴 异常 — 广告/SEO农场/软文嫌疑
• ⚪ 未验证 — 无法判断

🔒 4种验证模式
• 纯本地 — 零网络请求，零隐私风险，开箱即用
• 智能切换（推荐）— 轻量走本地，复杂走AI
• 仅云端 — 所有验证走AI模型
• 自建Docker — 在自己的电脑或NAS上搭建本地AI模型，完全私有

🆓 免费AI模型（推荐）
🌟 智谱GLM-4-Flash 永久免费，零门槛入门！另有 DeepSeek、通义千问、硅基流动、Groq、Moonshot、MiniMax、零一万物、百度千帆，均有免费额度。

💎 付费模型（更高精度）
OpenAI GPT-4o、Anthropic Claude-4、Google Gemini-2.5-Pro，以及字节豆包、腾讯混元、讯飞星火等。DeepSeek/通义千问/Moonshot等亦有付费高精度模型。

🐳 自建Docker（高级用户）
有GPU和Docker环境？可在自己的电脑或NAS上搭建本地AI模型，完全私有，数据不出你的网络。支持Ollama + Qwen2.5等开源模型。

🌐 双语翻译（应急）
外文页面一键双语对照翻译，支持联合国6大工作语言互译（中/英/法/西/俄/阿）。⚠️ 如网站有官方语言切换，请优先使用，本翻译仅为应急用途。

⚡ 快速开始
1. 安装后访问任意搜索引擎
2. 搜索结果自动标注颜色条
3. 鼠标悬停查看详细评分和原因
4. 想要更高精度？在设置页选择免费AI模型即可

🛡️ 隐私保护
• 本地模式：零数据收集，零网络请求
• 云端模式：仅发送搜索结果文本到您自行选择的AI提供商
• 自建Docker：数据完全不出你的网络
• API密钥仅存储在浏览器本地，我们绝不收集
