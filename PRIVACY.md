# VerityLens Privacy Policy / 隐私政策

**Last Updated / 最后更新**: 2026-06-22  
**Effective Date / 生效日期**: 2026-06-22

---

## English

### Overview

VerityLens ("we", "our", "the extension") is committed to protecting your privacy. This policy explains what data we collect, how we use it, and your choices.

### Data We Collect

**We collect ZERO data.** VerityLens does not:

- Collect, store, or transmit any personal information to our servers
- Track your browsing history or search queries
- Send your data to third-party analytics services
- Use cookies or any tracking technologies
- Require any account registration

### Data Stored Locally

The following data is stored **only on your device** using browser local storage:

| Data | Purpose | Stored By |
|------|---------|-----------|
| Channel mode preference | Remember your verification mode (local/smart/cloud/docker) | Browser storage |
| API provider selection | Remember which AI provider you selected | Browser storage |
| API key | Connect to your chosen AI provider (only if you configure one) | Browser storage |
| Confidence threshold | Your preferred verification sensitivity | Browser storage |
| Language preference | UI language (Chinese/English) | Browser storage |

**You can delete this data at any time** by removing the extension or clearing browser storage.

### Third-Party Services

If you choose to use cloud verification mode, your search result text may be sent to AI providers **that you explicitly configure**:

- You must manually enter your own API key
- We never provide, sell, or share API keys
- Data sent to AI providers is governed by each provider's own privacy policy
- All API calls are made directly from your browser — we never act as an intermediary

Supported providers (optional, user-configured only): DeepSeek, Zhipu GLM, Tongyi Qianwen, SiliconFlow, Groq, Moonshot, MiniMax, Yi, Baidu Qianfan.

### Local-Only Mode

VerityLens works fully in local-only mode with **zero network requests**. No data leaves your device. This is the default mode for users who prefer maximum privacy.

### Children's Privacy

VerityLens does not knowingly collect data from children under 13.

### Changes to This Policy

We may update this policy occasionally. Changes will be reflected in the "Last Updated" date above.

### Contact

For privacy questions: Open an issue at https://github.com/hannanlsa/VerityLens

## 中文

### 概述

VerityLens（"我们"、"本扩展"）致力于保护您的隐私。本政策说明我们收集哪些数据、如何使用以及您的选择。

### 我们收集的数据

**我们收集零数据。** VerityLens 不会：

- 收集、存储或传输任何个人信息到我们的服务器
- 追踪您的浏览历史或搜索查询
- 将您的数据发送到第三方分析服务
- 使用 Cookie 或任何追踪技术
- 要求任何账户注册

### 本地存储的数据

以下数据**仅存储在您的设备上**，使用浏览器本地存储：

| 数据 | 用途 | 存储方式 |
|------|------|----------|
| 通道模式偏好 | 记住您的验证模式（本地/智能/云端/Docker） | 浏览器存储 |
| AI 提供商选择 | 记住您选择的 AI 提供商 | 浏览器存储 |
| API 密钥 | 连接您选择的 AI 提供商（仅在您配置时） | 浏览器存储 |
| 置信度阈值 | 您偏好的验证灵敏度 | 浏览器存储 |
| 语言偏好 | 界面语言（中文/英文） | 浏览器存储 |

**您可以随时删除这些数据**，只需移除扩展或清除浏览器存储。

### 第三方服务

如果您选择使用云端验证模式，搜索结果文本可能会发送到**您自行配置的** AI 提供商：

- 您必须手动输入自己的 API 密钥
- 我们绝不提供、出售或共享 API 密钥
- 发送到 AI 提供商的数据受各提供商自身隐私政策约束
- 所有 API 调用均直接从您的浏览器发起——我们绝不作为中间人

支持的提供商（可选，仅用户自行配置）：DeepSeek、智谱GLM、通义千问、硅基流动、Groq、Moonshot、MiniMax、零一万物、百度千帆。

### 纯本地模式

VerityLens 在纯本地模式下完全可用，**零网络请求**。没有任何数据离开您的设备。这是注重隐私的用户的默认模式。

### 儿童隐私

VerityLens 不会有意收集 13 岁以下儿童的数据。

### 政策变更

我们可能会偶尔更新本政策。变更将反映在上方的"最后更新"日期。

### 联系方式

隐私问题：在 https://github.com/hannanlsa/VerityLens 提交 Issue
