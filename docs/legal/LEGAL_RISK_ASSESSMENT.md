# VerityLens 法律风险评估

## 主人 14:32 核心问题

> （此处是否有法律风险？）

## 结论：✅ **整体可控**（中低风险）

## 8 维度评估

### 1. 反不正当竞争法 ⚠️ 高风险 → ✅ 缓解
- **风险**：搜索引擎可能起诉"破坏商业模式"
- **缓解**：不抓取不排名不复制 = 只做标注 + 跳转辅助
- **判例**：Adblock 已被全球法院判合法

### 2. 广告法 ⚠️ 中风险 → ✅ 缓解
- **风险**：标注"虚假广告" = 商家起诉
- **缓解**：不标注"虚假" = 只标注"广告/SEO 农场"

### 3. 数据安全法 ⚠️ 中风险 → ✅ 缓解
- **风险**：抓取用户数据
- **缓解**：不抓取 = 全本地处理

### 4. 个人信息保护法 ✅ 低风险
- **风险**：收集用户搜索记录
- **缓解**：不收集 = opt-in

### 5. 爬虫协议 ⚠️ 中风险 → ✅ 缓解
- **风险**：爬搜索结果 = 违反 robots.txt
- **缓解**：不爬 = 用浏览器 DOM 注入

### 6. 商标法 ✅ 低风险
- **风险**：使用 360/百度/搜狗商标
- **缓解**：不使用

### 7. 著作权法 ✅ 低风险
- **风险**：复制搜索结果
- **缓解**：不复制 = 只显示元数据

### 8. 平台协议 ⚠️ 高风险 → ✅ 缓解
- **风险**：Chrome Web Store / 360 应用商店下架
- **缓解**：用户可手动装
