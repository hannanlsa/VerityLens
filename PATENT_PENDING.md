# VerityLens · 原创技术说明

## 核心原创算法

### 1. 跨模态自校验框架
- ASR（语音）+ OCR（屏幕）+ 文本 三元组交叉验证
- 5级置信度评分（high / medium / abnormal / partial_X / unverified）
- 统一时间轴 + 实体宇宙交叉引用

### 2. 搜索结果可信度标注
- DOM注入（非爬取）
- 3级颜色编码（绿 / 黄 / 红）
- 悬停弹窗显示可信度评分
- "跳转到真实来源"提示

### 3. 双语翻译置信度补位
- LLM自评 + 本地二次校验
- 数字/专有名词保留率检测
- 翻译置信度可视化标注
